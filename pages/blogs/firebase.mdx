**Firebase Firestore**

Now firebaes firestore is an **No-Sql** database which means is stores it's data in json Format ,just like mongodb stores it's data in BSON Format

Now firebase firestore can be directly connected through the web application without needing the server , **So it is called serverless Database** and can be connected through the backend server like **Node.js**
