

# Mongo DB


### Intro
Mongodb is opensource non sql database .It stores it’s data in JSON  like format  called BSON.
It’s easy to setup and it api works on node.js java c++ pythohn mostly all famous langaues

so Here is the step by step Guid how to set up a mongodb instances lin linux using command line and how to create ,drop database and insert and delte collections how to work with node.js  Mongoose api

### *Step 01* 
I’m using **pop os** (variant of Debian linux )

```
Sudo apt update
wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list
sudo apt-get update
sudo apt-get install -y mongodb-org
Sudo apt install vim 
```

##### commands explained



**sudo apt update** : to update all system packages     

**sudo apt install vim** : now vim is a text editor (we are using it to change mongodb.config files)
